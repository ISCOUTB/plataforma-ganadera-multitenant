# farmlink_app

Plataforma Móvil Multitenant para Gestión Integral de Fincas Ganaderas

Proyecto académico desarrollado en el marco de la asignatura Proyecto de Ingeniería 1 de la Universidad Tecnológica de Bolívar.

Actualmente en fase de desarrollo.



Descripción General

La Plataforma Móvil Multitenant para Gestión Integral de Fincas Ganaderas es una solución tecnológica orientada a la digitalización y centralización de procesos administrativos del sector ganadero.

El sistema está diseñado bajo una arquitectura multitenant, permitiendo que múltiples fincas operen de manera independiente dentro de una misma infraestructura tecnológica.
